astralCheckbox demo page

A really simple jQuery plugin that allows you to style a checkbox.

How to use

first include jQuery

<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js"></script>

than include astralCheckbox plugin

<script type="text/javascript" src="jquery.astralcheckbox.js"></script>

and call the astralCheckbox on a checkbox you wish to style

$(document).ready(function() {
	$('#some-checkbox').astralCheckbox();
});

The Logic

  1. First it hides the original checkbox.
  2. Than it creates a replacement markup for you to style and adds it before the original checkbox in the DOM tree.
  3. Markup consists of a wrapper div and an anchor inside.
  4. Anchor listens to click event and checks/unchecks the original checkbox accordingly.
  5. Anchor toggles active class to change the style state.

The defaults

There are only 3 options in astralCheckbox


wrapperClass: "astral-checkbox-wrapper",
anchorClass: "astral-checkbox-anchor",
triggerEventName: "astral-checkbox-change"

wrapperClass: defines the css class for div wrapper

anchorClass: defines the css class for the anchor

triggerEventName: defines the name of the callback event on each checkbox state change
